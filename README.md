# Influencer Campaign Performance Dashboard

## Project Overview

The **Influencer Campaign Performance Dashboard** is a data analytics project designed to evaluate influencer marketing campaigns using Python and Power BI. The dashboard helps marketing teams analyze campaign performance, engagement, audience behavior, return on investment (ROI), and overall campaign effectiveness to support data-driven decision-making.

---

## Objective

Build an interactive dashboard to:

* Analyze influencer campaign performance.
* Measure engagement across different platforms.
* Understand audience behavior and conversion trends.
* Evaluate campaign ROI.
* Help marketing teams make smarter campaign decisions.

## Project Workflow

### Phase 1 – Data Collection

* Imported the influencer campaign dataset.
* Verified data structure and attributes.

### Phase 2 – Data Cleaning

* Removed duplicates.
* Handled missing values.
* Corrected data types.
* Prepared a clean dataset for analysis.

### Phase 3 – Exploratory Data Analysis

* Analyzed campaign performance.
* Identified top-performing influencers.
* Compared engagement across platforms.
* Evaluated campaign ROI.
* Studied audience behavior.

### Phase 4 – Influencer Segmentation

* Segmented influencers into:

  * Nano
  * Micro
  * Macro
  * Mega
* Created ROI categories.
* Created Engagement categories.
* Calculated Cost per Engagement.

### Phase 5 – Dashboard Development

Created four interactive dashboard pages:

* Executive Overview
* Influencer Analytics
* Audience & Platform Analytics
* Financial Dashboard

* Spend vs ROI

---

## Key Insights

* Micro influencers achieved strong ROI.
* Platform performance varied based on engagement.
* Higher campaign spending did not always produce higher ROI.
* Audience groups showed different conversion behaviors.
* Cost-efficient campaigns generated better marketing returns.
